MY GAMES

Overview

The website MY GAMES is a personal gaming portal showcasing a curated collection of browser-based games. Designed by Sasank using HTML and CSS, it offers a user-friendly interface with a grid layout for easy navigation.

Key Features

  • Car Game: An action-packed racing game.
  • Shangchi Game: A martial arts-themed game.
  • RS Game: A shooting game.
  • Snake & Ladders: A classic board game.
  • Bubble Shooter: A puzzle game involving shooting bubbles.
  • Jet Fighter: A desktop-compatible aerial combat game.
  • KABIR VS TIGER 3D: A 3D battle game.
  • HIGHWAY FURY (MOTO RACING): A motorcycle racing game.
  • DARK-ROAD RIDERS_OF_DOOM: An action-adventure game.
  • SALAAR, OG, GLADIATOR: Upcoming games.

Design

The website uses clean, modern HTML and CSS for a responsive design, ensuring compatibility across devices.

Accessibility

All games are playable directly in the browser, eliminating the need for downloads or installations.
