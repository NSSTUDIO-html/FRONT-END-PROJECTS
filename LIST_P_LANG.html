Types of Languages in Computer Science

A complete classification of computer-related languages with examples

1. Machine-Level Language

Binary (`0` and `1`) directly understood by hardware.

Example: 10101010 11001100

2. Assembly Language

Uses mnemonics instead of binary. Needs an assembler.

Example: MOV A, B

4. Markup Languages

For structuring/presenting data.

Examples: HTML, XML, Markdown

5. Query Languages

Used for databases.

Examples: SQL, GraphQL, XQuery

6. Domain-Specific Languages

Special-purpose for specific fields.

Examples: MATLAB, R, Verilog, CSS

7. Modeling & Specification Languages

For system design and formal descriptions.

Examples: UML, Z notation

8. Hardware Description Languages

Used to design/simulate circuits.

Examples: Verilog, VHDL

9. NLP-Oriented Languages

Bridge between human languages & machines.

Examples: AIML, Inform

10. Esoteric Languages

Made for experiments, challenges, or fun.

Examples: Brainfuck, Whitespace
